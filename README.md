# Alice's Portfolio Website

一个优雅的个人作品集网站，使用 Next.js 13+ 和 TypeScript 构建。

## 技术栈

- **框架**: Next.js 13+ (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **字体**: Fiolex Girls (自定义字体)

## 本地开发

1. 克隆项目
```bash
git clone [repository-url]
```

2. 安装依赖
```bash
npm install
# 或
yarn install
```

3. 启动开发服务器
```bash
npm run dev
# 或
yarn dev
```

4. 构建生产版本
```bash
npm run build
# 或
yarn build
```

## 部署

项目可以轻松部署到 Vercel 平台：

1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成
